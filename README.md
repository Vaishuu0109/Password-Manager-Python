# Password Manager using Python

## Description
A secure local password manager built using Python that encrypts and stores credentials on disk using AES-based encryption.

## Features
- Master password protection
- AES encryption (Fernet)
- Add, view, delete passwords
- Encrypted local storage
- Secure key derivation using PBKDF2

## How to Run
1. Open Command Prompt
2. Navigate to project folder
3. Run:
   python password_manager.py

## Technologies Used
- Python
- Cryptography
- AES Encryption
- PBKDF2

## Project Structure
- password_manager.py
- vault.enc
